#Download VirtualBox

https://www.virtualbox.org/wiki/Downloads

Select Windows hosts link

#Download Ubuntu

https://ubuntu.com/download

Test or educational:
	Ubuntu Desktop
	https://ubuntu.com/download/desktop
	Click Download
	
Production:
	Ubuntu Server
	
#Install VirtualBox

Run script

#Create Virtual Machine

	Click on New icon
		Specify VM name & VM type
		Specify VM Ram
		Specify Hard disk: Create a virtual hard disk now
		Use VDI hard disk type
		Select size hard disk
		
	Click on Setting icon & go to Storage
		click on Empty disk icon & choose disk file..
		Select ubuntu iso file
	
	Click start icon
		Select statup disk:
			on scroll component, select ubuntu iso image (image download)
		Click start to start up the VM	
		
		
	To make the windows full screen:
		WE need to install Guest additions, to do that:
			Select Devices-> Insert Guess additions CD images...
			 Select Run
			 
	Reboot VM
